Cascading Style Sheets

¿Que es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación y el diseño de un documento HTML. Mientras que HTML se encarga de la estructura y el contenido de una página web, CSS se encarga de cómo ese contenido se verá y se sentirá en términos de diseño visual.

¿Que hace CSS?

CSS te permite controlar varios aspectos del diseño de una página web, tales como:

  • Colores: De texto, fondo, bordes, etc.
  • Fuentes: Tipo de letra, tamaño, estilo, peso, etc.
  • Espaciado: Margen, relleno, espacio entre letras, líneas, etc.
  • Layout (disposición): Posicionamiento de elementos, diseño en cuadrícula (grid), flexbox, etc.
  • Decoraciones: Sombras, bordes redondeados, gradientes, etc.
  • Resposive Design: Adaptar el diseño a diferentes tamaños de pantalla (móviles, tablets, etc.).

Estructura Basica de CSS

Un archivo CSS está compuesto por una serie de reglas. Cada regla tiene un selector el cual indica que elementos de HTML seran afectados y un bloque de declaraciones que contiene las propiedades y valores que se aplicaran a los elementos seleccionados.

Ventajas de usar CSS

  • Separación de contenido y diseño

    Mantener el HTML para la estructura y el contenido, y el CSS para el diseño, facilita la gestión y mantenimiento del sitio.

  • Reutilizacion

    Puedes aplicar el mismo archivo CSS a múltiples páginas HTML, logrando consistencia en todo el sitio.

  • Facilidad de Actualizacion

    Cambiando una sola línea en el archivo CSS, puedes modificar el diseño de múltiples páginas.
