Que es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Se utiliza para definir la estructura básica de un sitio web, incluyendo elementos como títulos, párrafos, enlaces, imágenes, tablas, formularios y otros componentes.
HTML no es un lenguaje de programación; en cambio, es un lenguaje de marcado, lo que significa que utiliza etiquetas para anotar y definir partes del contenido en una página web. Estas etiquetas se colocan entre corchetes angulares y suelen venir en pares, con una etiqueta de apertura y una etiqueta de cierre.

Como comenzar un documento HTML

  1. Crear un archivo HTML

    Abre un editor de texto (como Visual Studio Code, Sublime Text, o incluso el Bloc de notas).

    Crea un nuevo archivo y guárdalo con la extensión .html. Por ejemplo, index.html.

  2. Escribir la estructura basica

    Dentro de la estructura basica tenemos varias secciones importantes como: DOCTYPE,etiqueta head, etiquetas de metadatos, body

  3. Agregar contenido al cuerpo

    Debemos agregar dentro de nuestro body encabezados, parrafos, imagenes, etc.

  4. Guardar y visualizar la pagina

    Guarda el archivo después de agregar el contenido.

    Abre el archivo .html en un navegador web (puedes hacer doble clic en el archivo o abrirlo desde el navegador).

  5. Agregar mas contenido y elementos

    A medida que aprendas más sobre HTML, puedes agregar diferentes tipos de elementos, como listas, tablas, imágenes, formularios, etc.
